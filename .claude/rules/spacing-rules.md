# Spacing Rules

## Base Grid
8px base unit. All spacing values must be multiples of 8.

## Scale
- 4px  — hairline (borders, icon gaps)
- 8px  — tight (inline elements, badges)
- 16px — base (component padding)
- 24px — comfortable (card inner padding)
- 32px — section divider
- 48px — component groups
- 64px — section spacing (mobile)
- 96px — section spacing (desktop)
- 128px — hero spacing
- 192px+ — cinematic breathing room

## Rules
- Sections require minimum 96px vertical padding on desktop
- Use asymmetric vertical padding for editorial feel (top ≠ bottom)
- Maintain consistent inner padding per component type
- Content max-width: 1280px (layout), 768px (prose), 1024px (dashboard)
- Horizontal page padding: 24px mobile, 48px tablet, 80px desktop

## Avoid
- Pixel-perfect but visually cramped layouts
- Uniform spacing that creates flat visual rhythm
- Negative margins as a primary layout tool
