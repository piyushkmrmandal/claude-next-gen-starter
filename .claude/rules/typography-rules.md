# Typography Rules

## Scale
- Display: 72–120px, tight tracking (-0.04em to -0.06em)
- H1: 48–64px, tight tracking (-0.03em)
- H2: 36–48px, slightly tight (-0.02em)
- H3: 24–32px, normal to slightly tight
- Body: 16–18px, comfortable line-height (1.6–1.75)
- Caption: 12–14px, slightly loose tracking (0.01em)

## Weights
- Headings: 700–900 (bold to black)
- Body: 400–500 (regular to medium)
- Labels/UI: 500–600 (medium to semibold)

## Rules
- Always set `font-feature-settings: "kern" 1, "liga" 1`
- Use `text-rendering: optimizeLegibility` on headings
- Maintain minimum 4.5:1 contrast ratio (WCAG AA)
- Never use more than 2 typefaces per project
- Heading line-height: 1.0–1.2
- Body max-width: 65–75 characters (ch units)

## Avoid
- Random font mixing
- All-caps body text
- Centered body paragraphs longer than 2 lines
- Light weights (< 300) below 18px
