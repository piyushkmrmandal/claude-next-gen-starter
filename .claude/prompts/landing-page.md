# Landing Page Prompt

Generate a cinematic, conversion-focused landing page with:

## Structure
- Hero: full-viewport, asymmetric, strong typographic anchor
- Social proof: editorial layout, not a testimonial grid
- Features: storytelling sections, not feature cards
- CTA: singular, weighted, high-contrast

## Design Rules
- No generic startup hero illustrations
- No gradient blob backgrounds
- No feature card grids
- Use intentional whitespace as a design element
- Typography leads the visual hierarchy

## Motion
- Entrance: stagger reveal on scroll
- Hero: subtle parallax depth
- CTA: magnetic hover with spring physics
