# SaaS App Prompt

Generate a production-grade SaaS interface with:

## Layout
- Sidebar navigation: collapsible, icon + label, subtle active states
- Content area: full-bleed with inner max-width
- Data tables: high information density, clean row hover
- Empty states: illustrated, actionable

## Design Rules
- No generic dashboard card grids
- Data visualization with intentional color encoding
- Keyboard navigation and focus management
- Responsive across all breakpoints

## Motion
- Route transitions: fade + translate
- Loading states: skeleton with shimmer
- Micro-interactions on data mutation
