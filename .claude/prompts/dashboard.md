# Dashboard Prompt

Generate an analytics dashboard with:

## Layout
- Top-level KPI strip: 4 metrics, clean delta indicators
- Primary chart: full-width, interactive, brush-selectable
- Secondary charts: 2-column asymmetric grid
- Activity feed: chronological, event-typed

## Design Rules
- Chart colors: semantic, not decorative
- No donut charts for primary KPIs
- Consistent number formatting
- Accessible color contrast on all data encodings

## Motion
- Chart entrance: staggered bar/line draw
- KPI change: count-up animation
- Filter change: crossfade transition
