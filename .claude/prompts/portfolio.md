# Portfolio Prompt

Generate a cinematic personal portfolio with:

## Structure
- Intro: editorial, typographic-led, no photo headshots
- Work: case study cards with hover reveal
- About: asymmetric layout, editorial imagery
- Contact: minimal, single CTA

## Design Rules
- Strong typographic personality
- Dark mode by default
- Project imagery full-bleed behind typography
- No timeline layouts

## Motion
- Cursor: custom with magnetic elements
- Case study hover: clip-path reveal
- Page transitions: seamless morphing
