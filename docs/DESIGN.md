# DESIGN SYSTEM

# SPACING

Use 8px spacing grid.

# TYPOGRAPHY

Headings:
- bold
- tight tracking
- strong contrast

Body:
- high readability
- minimal clutter

# BORDER RADIUS

Cards:
- 24px large
- 16px medium

Buttons:
- pill radius preferred

# LAYOUTS

Prefer:
- asymmetry
- storytelling layouts
- layered sections
- immersive spacing

Avoid:
- centered SaaS clones
- repetitive grids
