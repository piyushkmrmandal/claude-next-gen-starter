# MOTION SYSTEM

Preferred easing:
cubic-bezier(0.22, 1, 0.36, 1)

Animation duration:
200ms–1200ms

Rules:
- Motion must feel weighted
- Use layered transitions
- Prefer opacity + translate
- Use cinematic staggering
- Add subtle parallax only

Avoid:
- random scaling
- excessive blur
- distracting motion
- floating effects
